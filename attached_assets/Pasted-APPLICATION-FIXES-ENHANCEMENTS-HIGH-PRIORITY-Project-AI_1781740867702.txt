3. RAPID MODE REBUILD

=====================



Current Issue:

Rapid Mode waits for processing before next capture.



This defeats the purpose of Rapid Mode.



Required Workflow:



Capture Card

↓

Image Stored Locally

↓

Immediately Return To Camera

↓

User Captures Next Card

↓

OCR Runs In Background

↓

Records Saved Automatically



Requirements:



* Unlimited continuous capture

* No waiting

* No save button

* No OCR blocking UI



Acceptance Criteria:



* User can scan 100 cards continuously.

* OCR processing must not interrupt capture flow.

Required Workflow:



Batch Mode

↓

Acceptance Criteria:
